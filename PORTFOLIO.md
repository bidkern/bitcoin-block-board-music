# Portfolio Overview: Bitcoin Block Board Music

Bitcoin Block Board Music is a browser-based data-mapping project that turns block structure into deterministic audio and visuals. It is included in the portfolio as creative technical depth, not as a claim that Jesse is targeting crypto, finance, or senior engineering roles.

## What this project proves

- Can map structured data into a repeatable output system.
- Can build an interactive browser experience with JavaScript, Web Audio, and inspectable UI controls.
- Can document how inputs become outputs so the project is understandable to a reviewer.
- Can package a project with a public demo, local run path, screenshots, and verification commands.

## Reviewer path

1. Open the public demo linked in [README.md](README.md).
2. Review the "How Bitcoin block data maps to sound" section.
3. Open [ARCHITECTURE.md](ARCHITECTURE.md) for the pipeline explanation.
4. Run `npm install --dry-run --ignore-scripts --no-audit --no-fund` and the `node --check` commands from the README.
